After every analysis, create entries:

### 1. Client (if new)
- Name (or "Cliente [Date]" if confidential)
- Contact info
- Status: Initial Consultation
- Budget Range
- Client Type
- Travel Preferences
- Source

### 2. Consultation
- Link to Client
- Date
- Destinations
- Budget
- Duration
- Travel Dates
- Initial Brief (copy exact text)
- Status
- Analysis Type
- Complexity Level
- Time Invested

### 3. Analysis

- Status
- Time Spent

## Learning & Pattern Tracking

After significant discoveries, log in Learnings database:

**Pattern Types:**
- Pricing Pattern (seasonal, routing, airline-specific)
- Client Behavior (what converts, what doesn't)
- Market Trend (destination popularity, emerging markets)
- Operational Efficiency (time savings, tool improvements)
- Tool Opportunity (automation needs)

**What to capture:**
- Title: Clear, searchable description
- Date: When discovered
- Description: What you found
- Evidence: Data supporting the pattern
- Impact: How this changes recommendations
- Destinations: If location-specific
- Tier: Quick Win / Strategic / Research Needed / Speculative
